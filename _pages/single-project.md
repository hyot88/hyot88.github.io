- ASAP (진행중)
  - 미션 계획(1Day, 3Day, 5Day, 7Day)을 세우고 달성하기 위해, 동기부여를 만들어주는 사이트 제작
  - 사용 기술
    - Spring Boot 2.1.7
    - JAVA 11
    - JPA
    - JWT 0.9.1
    - OAuth2 (google, naver, kakao)
    - Swagger 2.9.2
    - mustache
    - jquery, javascript
    - h2 database
    - MySQL
    - AWS (EC2, RDS, IAM, S3, CodeDeploy)
    - Travis CI
    - Jenkins
    - NGINX 1.20
  - Git 주소
    - <https://github.com/hyot88/ASAP>
    - <https://github.com/hyot88/ASAP-api>
